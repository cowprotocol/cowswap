## App commands
- Start dev server: `yarn start` or `yarn start:cowswap`
- Build: `yarn build:cowswap`
- Preview: `yarn preview`
- Lint: `yarn nx run cowswap-frontend:lint`
- Test: `yarn nx run cowswap-frontend:test`
- i18n (app-only): `yarn nx run cowswap-frontend:i18n`
- Cosmos: `yarn start:cosmos`
